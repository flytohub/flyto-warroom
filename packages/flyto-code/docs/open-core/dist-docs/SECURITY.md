# Security Policy

Report vulnerabilities **privately** to the Flyto2 maintainers. Do not open a
public issue and do not publish exploit details before coordinated disclosure.

- Preferred: GitHub private vulnerability reporting (Security → Report a
  vulnerability) on this repository.
- We aim to acknowledge within 48 hours and provide a status update within 7
  days.

Because this is a generated open-core mirror, a confirmed fix is applied
upstream and lands here on the next export.
