# Public Packages

This directory contains allowlisted public packages exported into Flyto2
Warroom CE:

- `flyto-core`: YAML runtime, browser automation, deterministic verification,
  and module SDK.
- `flyto-indexer`: local code intelligence, impact graph, SAST/SCA/secrets/IaC,
  and evidence gates.
- `flyto-i18n`: shared locale source and generated distributions.
- `flyto-code`: Warroom cockpit frontend.
- `flyto-contracts`: public OpenAPI, capability, evidence, scanner, runner,
  and product-verification contracts.

Private engine internals, commercial datasets, live remediation workers,
billing mutation, and Enterprise deployment code are not exported here.
